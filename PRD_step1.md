# PRD Step 1 — 냉장고 이미지 인식

## 목표
사용자가 냉장고 사진을 업로드하면 AI가 식재료 목록을 추출한다.

## 사용 모델
`nvidia/nemotron-nano-12b-v2-vl:free` (OpenRouter)

---

## 기능 요구사항

### 이미지 업로드
- 드래그 앤 드롭 또는 파일 선택으로 이미지 업로드
- 지원 포맷: JPEG, PNG, WebP
- 최대 파일 크기: 10MB
- 업로드 후 미리보기 표시

### 재료 인식
- 이미지를 OpenRouter API에 전달하여 식재료 목록 추출
- 프롬프트: 냉장고 안에 보이는 식재료를 JSON 배열로 반환하도록 지시
- 반환 형식 예시:
  ```json
  {
    "ingredients": ["달걀", "우유", "당근", "양파", "두부"]
  }
  ```
- 인식 결과를 화면에 태그(chip) 형태로 표시
- 사용자가 태그를 수동으로 추가/삭제 가능

### 진행 상태
- API 호출 중 로딩 스피너 표시
- 오류 발생 시 재시도 버튼 제공

---

## API 연동

- **엔드포인트**: `POST /api/analyze`
- **요청**: multipart/form-data (이미지 파일)
- **응답**:
  ```json
  { "ingredients": ["달걀", "우유", "당근"] }
  ```
- 이미지는 서버에서 base64 변환 후 OpenRouter에 전달
